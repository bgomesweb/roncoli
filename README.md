# 🏢 Roncoli

Este repositório contém o código-fonte do site oficial da **Roncoli**:
👉 https://roncoli.com.br/

---

## 🚀 Sobre o projeto

O projeto foi desenvolvido com foco em performance, identidade visual e experiência do usuário, representando a presença digital da marca Roncoli de forma moderna e eficiente.

---

## 🛠️ Tecnologias utilizadas

* **PHP 7.5**
* **Tailwind CSS**
* **HTML5**
* **JavaScript**

---

## 🎯 Objetivo

Desenvolver uma plataforma institucional sólida e responsiva, com foco em:

* Apresentação clara dos serviços e produtos
* Navegação intuitiva
* Layout moderno e alinhado com a identidade da marca
* Alta performance e carregamento otimizado
